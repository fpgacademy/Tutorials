Detailed Description

The header file includes for the linux pci driver.

This file defines the name of the driver, initialization/cleanup functions for the module, PCI device structure, file operations, and all the input parameters of the driver.

Remarks:
The file should not be included by any file other than "alt_up_pci_driver.c".

Initial value:

        { 0,},  { 0,}
}

The structure defines a list of the different type of PCI devices that the driver supports.

The content of the structure is zero at the beginning, and the IDs of the supported device will be added into this structure in function alt_up_pci_init() at the load time.

Remarks:
The last entry of the pci_id_table must be zero.
